# Flutter Expense Tracker

A Flutter application to manage daily expenses with local storage, filtering, and visual analysis.

## Features
- Add, delete, and manage daily expenses
- Filter transactions based on date/month
- Monthly expense analysis using pie chart
- Persistent local storage using Hive
- Clean and responsive UI

## Tech Stack
- Flutter
- Dart
- Riverpod (State Management)
- Hive (Local Database)

## How It Works
- Stores transaction data locally using Hive
- Filters transactions dynamically based on user selection
- Calculates monthly expenses and visualizes them using a pie chart
